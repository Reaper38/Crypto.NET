DESCRIPTION
-----------
This library is optimized for hashing (with MD5, SHA1, SHA256) a value x times. It is useful for e.g. key strengthening, where you have to hash one value many times.
For building this library you can use gcc, make sure you use the -O3 option during compiling for optimal speed.
